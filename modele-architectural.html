Visibilité entre les objets

Aptitude d'un objet à voir un autre objet et donc concrètement d'avoir une référence à un autre objet

Types de visibilité entre les objets

  • Visibilité d'attribut: B est un attribut de A
  • Visibilité de paramètre: B est fourni comme paramètre à une méthode de A
  • Visibilité locale: B est un objet local (et non un paramètre) dans méthode de A
  • Visibilité globale: B est visibile par tous les objets

Dépendance (couplage) entre les classes

Un classe est couplée à une autre classe ou en dépend

Types de dépendances entre les classes

  • Super-classe ou interface: Le fournisseur est une super-classe ou une interface
  • Attribut: Le fournisseur est un attribut de la classe
  • Méthode: Le fournisseur est un paramètre ou une valeur de retour d'une méthode
  • Variable: Le fournisseur est une variable utilisée localement dans une méthode

Distinguez dépendance (couplage) entre les classes et visibilité entre les objets (instances de classes)

  • Est-ce que la classe A est dépendante de la classe C?
  • Est-ce que l'objet a peut voir (accéder à) l'objet c?
